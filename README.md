# eBarber - Mobilna aplikacija za barberje




eBarber je mobilna aplikacija, namenjena digitalizaciji poslovanja frizerskih salonov. Omogoča enostavno upravljanje terminov in vključuje algoritem za razporejanje terminov.

Backend: ASP.NET Core 8, Entity Framework, SQL Server
Mobilna aplikacija: React Native (Expo)




cd Backend
dotnet restore
dotnet ef database update
dotnet run


cd MobileApp
npm install
npx expo start
